# ripple-tui

This is a small terminal UI project for learning [Rust] by building an ASCII-style
water ripple effect with [Ratatui].

[Rust]: https://rust-lang.org
[Ratatui]: https://ratatui.rs

Inspired by Hugo Elias's [2D water effect].

[2D water effect]: https://web.archive.org/web/19981202135801/http://freespace.virgin.net/hugo.elias/graphics/x_water.htm
